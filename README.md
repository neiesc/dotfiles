# dotfiles (GNU/Linux and WSL for Windows 10+)
## GNU/Linux distro Archcraft (based in ArchLinux)
1. `git clone --recurse-submodules -j8 https://github.com/neiesc/dotfiles.git`
2. Run `make`

### Other commands
- `git submodule update --remote`
