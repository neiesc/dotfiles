# dotfiles (GNU/Linux)
## GNU/Linux using Archcraft (Yet another minimal Linux distribution, based on Arch Linux.)
1. `git clone --recurse-submodules -j8 https://github.com/neiesc/dotfiles.git`
2. Run `make`

### Other commands
- `git submodule update --remote`
