# dotfiles (GNU/Linux)
## GNU/Linux using Archcraft (Yet another minimal Linux distribution, based on Arch Linux.)
- ```git clone --recurse-submodules -j8 https://github.com/neiesc/dotfiles.git```
- First change `export IS_HOME_PC=true/false`
- Run `sudo ./init.sh`
